第一个c语言 ​

c语言一定要有main函数,为主函数:

c
#include <stdio.h>

    printf("hello\n")
    //printf是一个库函数
    return 0
    //如果正常运行,就返回0,不正常就返回非零数值

//int是整型类型的意思

//快捷键:运行->ctrl+fn+f5
